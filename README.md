# 🎬 NCIP: Netflix Content Intelligence Platform

The **Netflix Content Intelligence Platform (NCIP)** is an elite data intelligence system that transforms raw streaming data into strategic business decisions. This "True Limit of EDA" edition pushes beyond descriptive tracking into **Prescriptive Intelligence**.

## 🚀 Strategic Vision
NCIP shifts from simple "data plots" to **Decision-Oriented Intelligence Systems**. It bridges the gap between raw entertainment data and platform strategy, enabling real-time exploration of global content trends with quantified reliability.

---

## 🧠 Intelligence Layers

### 1. Prescriptive Strategy Engine
- **Strategic Content Scoring**: A proprietary algorithm that ranks every title based on format (Movie/TV), audience rating (TV-MA dominance), and market recency.
- **Insight Confidence Tagging**: High/Medium/Low confidence markers on every insight, dynamically calculated based on data density.
- **Data Reliability Index**: A top-level KPI quantifying the trust-level of the underlying dataset (Cleanliness %).

### 2. Behavioral Intelligence
- **Content Lifecycle Analysis**: Measures the ratio of "New Releases" vs "Library Depth" (the gap between release year and platform add date).
- **Market Expansion Tracking**: Monitoring Netflix's aggressive geographical scaling by tracking unique country sourcing over time.

### 3. Structural Intelligence
- **Content Mix Evolution**: Visualizing the transition from traditional film share to binge-ready TV series dominance.
- **Genre Diversity Index**: A proprietary metric tracking the "variety breadth" of the Netflix catalog as it scales globally.

### 4. Audience & Outlier Intelligence
- **Opportunity Detection**: Identifying fast-growing emerging markets (countries with the most recent average content adds).
- **Anomalous Reporting**: Automated detection of "Extreme Durations" and data outliers.

## 🚀 Launching the Platform

### 1. Install Dependencies
Ensure you have the full intelligence stack:
```bash
pip install streamlit pandas plotly seaborn matplotlib
```

### 2. Run the Dashboard
Power up the NCIP platform locally:
```bash
streamlit run app.py
```

# 🎬 NCIP: Netflix Content Intelligence Platform

The **Netflix Content Intelligence Platform (NCIP)** is an elite data intelligence system that transforms raw streaming data into strategic business decisions. This version represents the **Absolute Machine Learning Ceiling**, featuring both Supervised (NLP) and Unsupervised (Clustering) models.

## 🚀 Strategic Vision
NCIP shifts from simple "data plots" to **AI-Integrated Intelligence Systems**. It bridges the gap between raw entertainment data and platform strategy, enabling real-time exploration of global content trends supported by multi-layer Machine Learning.

---

## 🧬 Unsupervised Learning Layer
### Market Segmentation & Archetyping
A high-sophistication module that uses **K-Means Clustering** to segment global markets into strategic archetypes.
- **Features**: Analyzes TV Show Ratio, Production Volume, Strategic Score, and Recency.
- **Archetypes**:
    - **Global Powerhouses**: Volume leaders (e.g., USA, India).
    - **Binge-Culture Hubs**: Countries focused on serial TV show production.
    - **Strategic High-Value Hubs**: Producers of high-recency, top-rated content.
    - **Emerging Specialists**: Late-entry, developing markets.
- **Visualization**: Features a 3D Interactive Market Cluster Plot for macroeconomic discovery.

---

## 🤖 Data Science & AI Layer
### SmartContent AI Recommender
A high-performance recommendation engine built using **Natural Language Processing (NLP)**.
- **Technology**: TF-IDF Vectorization and Cosine Similarity.
- **Interactivity**: Semantic search for titles with "Match Confidence %" calculation.

---

## 🧠 Prescriptive Strategy Engine
- **Strategic Content Scoring**: Ranking titles by format, audience rating, and market recency.
- **Insight Confidence Tagging**: High/Medium/Low markers calculated based on data density.
- **Data Reliability Index**: Quantifying trust-level of the underlying intelligence source.

---

## 🛠️ Technology Stack
- **AI/ML**: [Scikit-learn](https://scikit-learn.org/) (NLP, K-Means Clustering, StandardScaler)
- **UI Layer**: [Streamlit](https://streamlit.io/) (Premium Dark Mode)
- **Viz Engine**: [Plotly](https://plotly.com/python/) (Interactive 3D & 2D Graphics)
